# Seedance 视频提示词生成器

为字节跳动 **Seedance 2.0（即梦）** 平台生成专业的中文视频提示词。

## 功能特性

### 十大核心能力

| 能力 | 说明 |
|------|------|
| 纯文本生成 | 无需素材，文字描述生成视频 |
| 一致性控制 | 多镜头保持角色/产品统一 |
| 运镜与动作复刻 | 上传视频复刻镜头语言 |
| 创意模板/特效 | 模仿转场、广告、电影片段 |
| 剧情创作/补全 | 根据分镜生成完整剧情 |
| 视频延长 | 前向/后向平滑延长 |
| 声音控制 | 音色参考、对白、音效 |
| 一镜到底 | 连贯长镜头场景过渡 |
| 视频编辑 | 角色替换、剧情颠覆 |
| 音乐卡点 | 画面节奏匹配音乐节拍 |

## 使用方法

### 基本使用

```bash
python3 scripts/gen_prompt.py "仙侠战斗，15 秒，横屏"
```

### 带参考素材

```bash
python3 scripts/gen_prompt.py "产品展示广告" --images 1 --type product
```

### 超长视频

```bash
python3 scripts/gen_prompt.py "30 秒仙侠出场短片" --duration 30
```

## 平台规格

| 类型 | 规格 |
|------|------|
| 图片 | jpeg/png/webp，最多 9 张，<30MB |
| 视频 | mp4/mov，最多 3 个，2-15 秒，<50MB |
| 音频 | mp3/wav，最多 3 个，≤15 秒，<15MB |
| 文件上限 | 最多 12 个文件（合计） |
| 生成时长 | 4-15 秒 |
| 分辨率 | 支持 2K 输出 |

**⚠️ 重要提示**：不支持含写实真人脸部的素材！

## 使用示例

### 示例 1：纯文本生成

**输入**：
```
帮我写一段仙侠战斗的视频提示词，15 秒，横屏
```

**输出**：
```
15 秒仙侠高燃战斗镜头，金红暖色调，
0-3 秒：低角度特写主角蓝袍衣摆被热浪吹得猎猎飘动，
双手紧握雷纹巨剑，剑刃赤红电光持续爆闪...
4-8 秒：环绕摇镜快切，主角旋身挥剑...
9-12 秒：仰拍拉远定格慢放，主角跃起腾空...
13-15 秒：缓推特写主角落地收剑的姿态...
```

### 示例 2：多模态参考

**输入**：
```
我有一张可乐产品图，想做一个炫酷的产品展示广告
```

**输出**：
```
@图片 1 中的可口可乐饮料，360 度高速旋转 2 圈后，
突然停住蓄力分裂成了 3 个部分进行展示...
参考素材：@图片 1：可乐产品正面高清图
```

## 高级技巧

1. **时间戳分镜法** - 按秒精确控制（如 `0-3 秒：...`）
2. **技术参数指定** - 声明画幅比、帧率、色调
3. **禁止项声明** - 排除水印、字幕、Logo
4. **超长视频分段** - >15 秒自动拆分多段拼接
5. **台词用引号** - 标注角色名和情绪
