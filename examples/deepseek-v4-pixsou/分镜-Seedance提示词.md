# 🎬 《DeepSeek V4 + 像素灵境》Seedance 提示词

> 直接粘贴到即梦使用。2 段 × 15 秒，9:16 竖版，2K。

## 🎞️ 第 1 段（0-15 秒）· 钩子 + V4 数据

```
15秒电影级科技资讯开场，黑蓝主色调，体积光强烈。
0-3秒：全黑背景，中央两个抽象圆形 logo 图腾（一个代表 AI 模型，
一个代表创作工具）从屏幕两侧飞入，到中央交汇碰撞，
碰撞点爆出一圈蓝色光波扩散；
3-7秒：硬切。深蓝到青蓝渐变背景，3D 立体数据柱状图从底部升起，
左侧旧数值"58%"的柱状图碎裂消失，右侧新数值"70%"的柱状图
以冲击波方式炸开，伴随数字粒子爆裂；
7-11秒：环绕镜头360度展示 4 个数据柱状图依次升起
（1.6T 参数 / 49B 激活 / 1M 上下文 / 24元价位），
每个数据柱旁出现抽象的百分比浮动数字；
11-15秒：镜头突然拉远，所有数据柱压成一张数据卡片飞向画面深处，
过渡到下段。3D 渲染，玻璃质感，体积光强烈。
```

## 🎞️ 第 2 段（15-30 秒）· 像素灵境 + 时间轴 + 结尾

```
接续上段。15秒电影级科技资讯收尾，黑蓝主色调。
15-20秒：硬切。一个抽象人形剪影站在 3D 场景中央，镜头围绕剪影
360度缓慢环绕，剪影周围出现 3 个屏幕投影（视频画面/3D 场景/AI 创作），
时间感从慢到快，画面质感从粗糙到精致；
20-25秒：硬切。时间轴横跨屏幕，从左到右依次亮起 4 个里程碑节点
（2023 / 2024 / 2025 / 2026），每个节点亮起时伴随冲击波扩散，
最终在 2026 处汇聚成一个巨大的发光光点；
25-30秒：硬切。黑色背景中央一个抽象的 "iPhone 时刻" 符号
（苹果轮廓 + 闪光）缓慢旋转 90 度，镜头轻微推近，
底部出现"AI 视频的 iPhone 时刻"字样的视觉暗示（图形化处理）。
3D 渲染，玻璃质感，体积光。
```

## ⚠️ 品牌安全

- 所有"logo"用**抽象图形**（圆形/几何符号）代替
- 避开写实人脸
- 时间轴上的"年份"用纯数字
- 苹果轮廓用通用剪影，不出现咬了一口的苹果
